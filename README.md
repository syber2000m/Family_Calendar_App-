# 우리 가족 일정표 (Family Calendar App)

가족 구성원별로 일정을 등록하고 달력에서 한눈에 확인할 수 있는 웹 캘린더 앱입니다. 순수 HTML/CSS/JavaScript로 작성되었으며, 별도 서버 없이 브라우저에서 바로 실행할 수 있습니다.

## 주요 기능

- 📅 월간 달력 보기 (이전/다음 달 이동, 연/월 선택 팝업)
- 👨‍👩‍👧‍👦 가족 구성원(엄마, 아빠, 승우, 정우, 창우)별 일정 등록 및 색상 구분
- 다중 구성원 선택 지원 (한 일정에 여러 명 지정 가능)
- 하루짜리 일정과 기간(여러 날짜) 일정 등록
  - 달력에서 날짜 클릭 후 Ctrl+클릭으로 기간 지정
  - 날짜 더블클릭으로 빠른 일정 추가
- 일정 수정 / 삭제
- 구성원별 필터링으로 특정 가족 구성원의 일정만 보기
- 전체/완료 일정 개수 통계 표시
- `localStorage`를 이용한 브라우저 로컬 저장 (새로고침해도 데이터 유지)

## 실행 방법

별도의 빌드나 설치 과정 없이 `index.html`을 브라우저에서 열면 바로 사용할 수 있습니다.

```bash
# 예시: VSCode Live Server 등으로 실행하거나
# index.html 파일을 브라우저에 드래그 앤 드롭
```

## 파일 구성

| 파일 | 설명 |
| --- | --- |
| `index.html` | 앱의 마크업 구조 및 팝업(모달) 정의 |
| `style.css` | 전체 스타일 및 레이아웃 |
| `main.js` | 일정 데이터 관리, 달력 렌더링, 이벤트 처리 로직 |

## 데이터 저장

모든 일정은 브라우저의 `localStorage`에 `family-todo-app-data` 키로 저장됩니다. 별도의 백엔드나 데이터베이스는 사용하지 않습니다.
